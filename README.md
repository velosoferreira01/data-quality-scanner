# Universal Data Quality Scanner

Projeto para análise universal de qualidade de dados.

Suporta múltiplas fontes:

- CSV
- Excel
- Parquet
- DuckDB
- PostgreSQL

O pipeline executa:

1. Ingestão de dados
2. Data scan
3. Métricas de qualidade
4. Score por coluna
5. Score por tabela
6. Aplicação de regras
7. Geração de relatórios
